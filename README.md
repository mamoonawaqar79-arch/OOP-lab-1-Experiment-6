# Complex Number Operations in C++

## Description
A simple C++ program that performs arithmetic operations on complex numbers using Object-Oriented Programming (OOP). The program allows users to input two complex numbers and calculate their sum, difference, and product.

## Features
- Input complex numbers
- Display complex numbers
- Addition
- Subtraction
- Multiplication
- OOP-based implementation

## Concepts Used
- Classes and Objects
- Constructors
- Encapsulation
- Member Functions

## Sample Input
```text
3+2i
4+5i
```

## Sample Output
```text
c1 = 3+2i
c2 = 4+5i
Sum: 7+7i
Difference: -1-3i
Product: 2+23i
```

## How to Run
```bash
g++ ComplexNumber.cpp -o ComplexNumber
./ComplexNumber
```
